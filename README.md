# Partical-Gester-Controller
Application
The Partical Gesture Controller is a gesture-based control system that allows users to interact with devices using hand movements instead of physical buttons. The project uses sensors and a microcontroller to detect gestures and perform specific actions.

## 🎯 Features
- Detects hand gestures
- Controls devices without physical contact
- Easy-to-use interface
- Real-time gesture recognition
- Low-cost and efficient implementation

## 🛠️ Technologies Used
- Arduino UNO
- Arduino IDE
- Embedded C
- Ultrasonic Sensor / IR Sensor (if applicable)
- Servo Motor (if applicable)
- Breadboard & Jumper Wires

## ⚙️ Working Principle
1. The sensor detects hand movement.
2. The Arduino processes the sensor data.
3. The detected gesture is compared with predefined commands.
4. The corresponding action is executed.
